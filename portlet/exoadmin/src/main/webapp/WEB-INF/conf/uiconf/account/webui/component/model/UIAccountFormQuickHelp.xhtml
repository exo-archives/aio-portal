In order to create a new account, you need to enter the information in each tab on the left
  • Use the tab Account Templates to select a pre-configure account.
  • Use the tab Account Settings to enter your basic and required information
  • Use the tab User Profile to enter in detail your personal information
  • Use the tab User Membership to add the additional membership. Note that only the admin can add the membership

If you already have an account but forget the username or password, use the "forget username/password" form or contact the admin.

The account template is a pre-configure account with certain features and capabilities.
  • Guest Account: A basic account, the user has no right to modify the portal configuration
  • User Account: A normal user account, the user is allowed to modify the portal configuration
  • Admin Account: A power user account, The user has all the right
